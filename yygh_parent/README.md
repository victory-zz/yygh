技术栈

SpringBoot：简化新Spring应用的初始搭建以及开发过程

MyBatis-Plus：持久层框架
Redis：内存缓存
RabbitMQ：消息中间件
HTTPClient: Http协议客户端
Swagger2：Api接口文档工具
Nginx：负载均衡
Lombok
Mysql：关系型数据库
MongoDB：面向文档的NoSQL数据库

Vue.js：web 界面的渐进式框架
Node.js： JavaScript 运行环境
Axios：Axios 是一个基于 promise 的 HTTP 库
NPM：包管理器
Babel：转码器
Webpack：打包工具
其他
Docker ：容器技术
Git：代码管理工具
